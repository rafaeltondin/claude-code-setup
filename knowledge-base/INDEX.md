## CREDENCIAL VAULT

CLI: `node "~/.claude/task-scheduler/credential-cli.js"`
Comandos: `get <NAME>` | `list` | `has <NAME>` | `names-for <PREFIX>` | `run <script.js>`

| Prefixo | Credenciais |
|---------|-------------|
| FB_ | FB_ACCESS_TOKEN, FB_APP_ID, FB_APP_SECRET, FB_AD_ACCOUNT_ID |
| SHOPIFY_ | SHOPIFY_ACCESS_TOKEN |
| PINHA_ | PINHA_SHOPIFY_ACCESS_TOKEN, PINHA_SHOP_NAME |
| EVOLUTION_ | EVOLUTION_API_KEY, EVOLUTION_INSTANCE_TOKEN |
| GOOGLE_ | GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REFRESH_TOKEN |
| SULETIQUETAS_ | SULETIQUETAS_WC_CONSUMER_KEY, SULETIQUETAS_WC_CONSUMER_SECRET, SULETIQUETAS_WC_STORE_URL |
